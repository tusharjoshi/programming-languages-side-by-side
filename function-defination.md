# Function Defination and Call

## C

```c
// define a function
void some_function() {
    // some logic
}

// call the function
some_function();
```

## C\#

```csharp
public class Main {
    public static void Main(String[] args) {
        // call the function
        SomeFunction();
    }

    // define a function
    public static void SomeFunction() {
        // logic
    }
}
```
